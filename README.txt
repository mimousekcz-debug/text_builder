2. Nahraj vsechny soubory do noveho GitHub repozitare.
3. Ve Vercelu klikni Add New > Project.
4. Vyber repozitar.
5. Klikni Deploy.

KDYZ CHCES TESTOVAT NA PC:
1. Nainstaluj Node.js: https://nodejs.org
2. Otevri slozku projektu v terminalu.
3. Spust:
   npm install
   npm run dev
4. Otevri adresu, kterou terminal ukaze.

Aplikace uklada projekt do prohlizece tlacitkem Ulozit.
